# Wallet Setup

inkd bot supports two wallet modes. Use `/start` to get started.

---

## Bot-Managed Wallet (recommended)

Tap **🆕 New Wallet**. The bot generates an Ethereum wallet and stores the private key encrypted in your session (AES-256-GCM).

- You get the address and private key (shown **once** — save it immediately)
- The bot signs USDC payments on your behalf
- Full upload access (`/upload_text`, `/upload_repo`, file uploads)
- Send funds with `/withdraw`

### Fund your wallet

Send USDC to your address on Base. Minimum ~$0.20 USDC for uploads.

Check balance anytime:

```
/wallet
```

Shows ETH + USDC balance and a QR code for easy deposits.

### Send funds out

```
/withdraw
```

Choose USDC or ETH → enter recipient address → enter amount → confirm. One flow, no external app needed.

### Export your key

```
/export_key
```

Shows your private key after a confirmation step. Use this to import into MetaMask or any EVM wallet.

---

## Connected Wallet (read-only)

Tap **🔑 Connect Wallet**. The bot gives you a challenge message. Sign it with your Base wallet and send the signature back.

- Read-only: can view projects and history
- Cannot upload (bot needs the private key to sign USDC payments)

---

## Session storage

Sessions are stored in SQLite on the server. Your encrypted key is tied to your Telegram chat ID. No one else can access it.

---

## Security notes

- Private keys are encrypted with AES-256-GCM using a server-side secret
- Keys are never logged or transmitted outside the server
- `/export_key` requires a confirmation tap before displaying the key
- If you lose your Telegram access, use the exported private key to recover your wallet
