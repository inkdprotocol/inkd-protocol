# Projects & Discovery

## Your projects

```
/my_projects
```

Lists your most recent projects. Tap any project to open it.

## History

```
/history
```

Recent uploads with direct share links.

## Search

```
/search <name>
```

Search public projects by name. Returns results with **📂 Details** buttons.

## Trending

```
/top
```

Browse the latest public projects from all wallets.

## Protocol stats

```
/stats
```

Live stats: total project count, token supply, network.

---

## Project details

Tap any project to see:

- Owner address + version count
- **👁 Preview** — first 400 chars of the latest version (text files)
- **🆕 Push Version** — push a new version with auto-suggested tag
- **🔗 Share** — formatted share card with Arweave link
- **⬇ v1.0.0** — download any version directly

## Download

Tap **⬇ v1.x** on any version. The bot fetches the file from Arweave and sends it in chat. Files over 48 MB get a direct Arweave link.

## Preview

Tap **👁 Preview** to see the first 400 characters of a text file inline — without leaving the chat or opening Arweave.

## Share

Tap **🔗 Share** for a formatted card with project name, version count, Arweave link, and Basescan link. Forward it anywhere.

## Inline search

Type `@inkdbot <query>` in any Telegram chat to search inkd without opening the bot.
