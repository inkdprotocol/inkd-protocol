# Projects & Discovery

## My Files

Tap **📁 My Files** from the home screen to see all your uploaded projects. Each project shows its visibility (🌍 public / 🔒 private) and version count.

## Project details

Tap any project to open it:

- Owner address + version count
- **🆕 Push Version** — push a new version with auto-suggested tag
- **🔗 Share** — formatted share card with Arweave link
- **👁 Preview** — first 400 chars of the latest version (text files only)
- **⬇ v1.x** — download any version directly
- **🗑 Hide** — remove from My Files (archived, not deleted)

## Archiving projects

Tap **🗑 Hide** on any project to archive it. It disappears from My Files but stays on-chain forever.

To see archived projects, tap **📦 Archived** at the bottom of My Files. Tap **↩️ Restore** on any entry to bring it back.

## Share on X

After every successful upload, a **🐦 Share on X** button appears. Tap it to open X with a pre-filled tweet:

- **Public uploads** include a direct Arweave link to the file
- **Private uploads** include a CTA to try the bot

## Search

```
/search <name>
```

Search public projects by name.

## Trending

```
/top
```

Browse the latest public projects from all wallets.

## Protocol stats

```
/stats
```

Live stats: total project count, token supply, network.

---

## Download

Tap **⬇ v1.x** on any version. The bot fetches the file from Arweave and sends it in chat. Files over 48 MB get a direct Arweave link.

## Preview

Tap **👁 Preview** to see the first 400 characters of a text file inline — without leaving the chat.

## Inline search

Type `@inkdbot <query>` in any Telegram chat to search inkd without opening the bot.
