# Telegram Bot

[@inkdbot](https://t.me/inkdbot) is the easiest way to use inkd. No SDK, no CLI, no terminal. Just Telegram.

You send a file or text. The bot uploads it to Arweave, registers it on Base, and charges your wallet in USDC — all in one flow.

---

## Commands

| Command | Description |
|---|---|
| `/start` | Set up wallet (skips straight to your wallet if already connected) |
| `/tutorial` | Interactive guided tour (4 steps) |
| `/wallet` | Show address, ETH & USDC balance, QR code |
| `/withdraw` | Send USDC or ETH to any address |
| `/export_key` | Export your private key (bot-managed wallets) |
| `/upload_text` | Upload text content to Arweave |
| `/upload_repo` | Archive a GitHub repo permanently |
| `/my_projects` | View your on-chain projects |
| `/search <name>` | Search public projects |
| `/history` | View your last 5 uploads |
| `/links` | Website, Twitter, buy $INKD |
| `/cancel` | Cancel current action |
| `/help` | Show all commands |

## Upload privacy

Every upload asks before proceeding:

```
🔓 Public   🔒 Private
✖️ Cancel
```

**Public** — content stored as-is on Arweave, anyone with the hash can read it.

**Private** — content encrypted with AES-256-GCM before upload. Only your wallet can decrypt. See [Private Uploads](/concepts/private-uploads).

---

## How it works

```
User → Telegram Bot
           │
           ├── /v1/upload  ──────────→ Arweave (permanent storage)
           ├── /v1/projects ─────────→ InkdRegistry on Base (x402 USDC)
           └── /v1/projects/*/versions → same
```

You pay in USDC on Base. The bot holds a bot-managed wallet for you (encrypted, AES-256-GCM). No custody risk — you own the key.
