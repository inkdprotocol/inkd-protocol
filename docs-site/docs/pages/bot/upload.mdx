# Uploading

Everything you store goes permanently to Arweave and is registered on-chain on Base. Paid in USDC from your bot wallet.

---

## How to upload

Tap **⬆️ Upload a File** from the home screen — or just send a file, photo, or video directly. The bot detects it automatically.

---

## File, Photo, or Video

Send any file, image, or video directly to the bot.

1. Bot receives it and asks for a name
2. Enter a name
3. Enter a description (optional — tap ⏭ Skip)
4. Choose visibility: **🌍 Public** or **🔒 Only me**
5. Confirm cost → stored

Supports all file types. Max 50 MB.

---

## Text / Note

Tap **📝 Text / Note** from the upload menu or use `/upload_text`.

1. Enter a name
2. Enter a description (optional)
3. Paste the text
4. Choose visibility
5. Confirm → stored

---

## GitHub Repo

Tap **🐙 GitHub Repo** or use `/upload_repo`.

1. Send a GitHub username, profile link, or repo:
   - `inkdprotocol` → lists all public repos with ⭐ stars
   - `github.com/inkdprotocol` → same
   - `owner/repo` → direct upload
   - `https://github.com/owner/repo` → direct upload
2. Bot downloads the ZIP, shows size + cost
3. Choose visibility
4. Confirm → stored

---

## Public vs Private

Before every upload you choose:

- **🌍 Public** — anyone with the Arweave link can read it
- **🔒 Only me** — AES-256 encrypted before upload. Only your wallet can decrypt.

See [Private Uploads](/concepts/private-uploads) for the full security model.

---

## Cost Estimator

Not sure what something will cost? Check before uploading:

```
/estimate 500kb   → $0.10 USDC
/estimate 4mb     → $0.14 USDC
/estimate 100mb   → $1.83 USDC
```

Exact cost is always shown before you confirm.

---

## Push a new version

Already have a project? Push an update:

1. Go to **📁 My Files** → tap the project
2. Tap **🆕 Push Version**
3. Bot suggests the next version tag automatically (e.g. v1.0.3 → v1.0.4)
4. Add changelog or skip
5. Choose content: text or GitHub repo
6. Confirm → new version pushed on-chain
